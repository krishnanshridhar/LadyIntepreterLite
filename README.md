# She Said What? v2

Two modes in one app — **😂 Comedy** for laughs, **🧠 Serious** for moments that actually matter.

## Quick start

```bash
pip install -r requirements.txt
ANTHROPIC_API_KEY=sk-ant-... streamlit run app.py
```

## Streamlit Cloud

Push `app.py`, `requirements.txt`, `README.md` to GitHub, connect at share.streamlit.io, then set your key under **Manage app → Secrets**:

```toml
ANTHROPIC_API_KEY = "sk-ant-your-key-here"
```

## Features

| 😂 Comedy | 🧠 Serious |
|---|---|
| 💬 What She Really Said | 🙏 Apology Workshop |
| 📖 The Phrase Bible | ⚖️ Argument Referee |
| 🚨 Danger Meter | 🎁 Gift Advisor |
| 🔢 Emoji Translator | 📅 Date Planner |
| 😶 Silence Decoder | 🌡️ Relationship Check-in |

## Personalise

Fill in the sidebar: her name, your name, relationship type, years together — every response is tailored to your context.
